AI Assist — Vercel Proxy Setup

AI Assist requires a Vercel serverless function to proxy Claude API calls. Create a file api/ai.js in your CRM GitHub repo with the code below, then add ANTHROPIC_KEY as a Vercel environment variable.
export default async function handler(req, res) { res.setHeader('Access-Control-Allow-Origin', '*'); if (req.method === 'OPTIONS') return res.status(200).end(); const r = await fetch('https://api.anthropic.com/v1/messages', { method: 'POST', headers: { 'Content-Type': 'application/json', 'x-api-key': process.env.ANTHROPIC_KEY, 'anthropic-version': '2023-06-01' }, body: JSON.stringify({ model: 'claude-sonnet-4-20250514', max_tokens: 1000, ...req.body }) }); const data = await r.json(); res.status(200).json(data); }
After adding api/ai.js to GitHub: Vercel → Project Settings → Environment Variables → add ANTHROPIC_KEY = your Claude API key
